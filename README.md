# TechNova Kurumsal Websitesi

TechNova Yazılım A.Ş.'nin profesyonel kurumsal web sitesi. Responsive tasarım, interaktif bileşenler ve modern teknolojilerle geliştirilmiştir.


---

## ✨ Öne Çıkan Özellikler
- **Otomatik Slider**: 4sn'de bir geçiş yapan görsel slider
- **Haber Modülleri**: "Devamını Oku" ile genişleyen haber kartları
- **İş Başvuru Sistemi**: Dinamik form validasyon ve dosya yükleme
- **Ekip Profilleri**: Hover efektli interaktif ekip üyesi kartları
- **Müşteri Yorumları**: Otomatik dönen testimonial karusel
- **İletişim Formu**: Anlık geri bildirimli AJAX entegrasyonu

---

## 🛠️ Teknoloji Yığını
| Teknoloji | Kullanım Amacı |
|-----------|----------------|
| Bootstrap 4.5 | Responsive grid ve navbar |
| JavaScript | Slider/Form fonksiyonları |
| jQuery | DOM manipülasyon |
| Font Awesome | İkon setleri |
| Google Maps API | Harita entegrasyonu |
